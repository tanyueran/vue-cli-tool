# 脚手架项目生成工具

## 说明

```text
使用此工具可以生成一个基于vue-cli@4的脚手架项目，里面包含了作者常用的一些东西。
```

## 模板项目依赖

```text
vue-cli@4
vue
vue-router
vuex
axios
scss
element-ui
Airbnb eslint
```

## 目录说明

```text
bin-- 入口执行文件
config -- 配置文件
lib -- 项目生成后，下载依赖
output -- 项目输出在这儿
template -- 模板项目源
```

## 使用

```text
node ./bin/init-project.js --init [projectName]
```
